CSS

(Cascading Style Sheets)

It is for styling this whole structure or the individual. Is works in conjunction with HTML, but is not HTML itself

CSS describes how HTML elements should be displayed.



Css Selectors Examples:


The .example_class Selects all elements with class="example_class"

The #example_id Selects the element with id="example_id"

The * Selects all elements



CSS Example

}
