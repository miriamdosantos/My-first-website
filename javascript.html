JavaScript

Javascript

It is the programming languages that runs in the browsers.

Modern JavaScript is a “safe” programming language. It does not provide low-level access to memory or CPU, because it was initially created for browsers which do not require it.

JavaScript’s capabilities greatly depend on the environment it’s running in. For instance, Node.js supports functions that allow JavaScript to read/write arbitrary files, perform network requests, etc.



In-browser JavaScript can do everything related to webpage manipulation, interaction with the user, and the webserver.

  • it used for adding responsiveness
  • allows designers to create interactive elements
  • Implement basic automation and give users feedback
  • Update Web page content on the fly
  • Communicate with the cloud (javascript + XML: Ajax)
